# @hermanos-ti/internal_lib

Bem-vindo à documentação da biblioteca de componentes internos da Hermanos TI.

## Sobre a Biblioteca

A `@hermanos-ti/internal_lib` é uma biblioteca de componentes React reutilizáveis desenvolvida para padronizar a interface e acelerar o desenvolvimento de aplicações internas.

## Instalação

```bash
npm install @hermanos-ti/internal_lib
```

## Uso Básico

```jsx
import { Button, Card, Input } from '@hermanos-ti/internal_lib';
import '@hermanos-ti/internal_lib/dist/style.css';

function App() {
  return (
    <div>
      <Button />
      <Card />
      <Input />
    </div>
  );
}
```

## Estrutura da Biblioteca

A biblioteca está organizada em duas categorias principais:

### Componentes

A biblioteca inclui 18 componentes prontos para uso:

- **ActionBar** - Barra de ações
- **Badge** - Badge/etiqueta
- **Breadcrumb** - Navegação breadcrumb
- **Button** - Botão
- **Card** - Card/container
- **Container** - Container
- **Details** - Detalhes
- **Drawer** - Gaveta lateral
- **Frame** - Frame/quadro
- **Header** - Cabeçalho
- **Input** - Campo de entrada
- **Menu** - Menu
- **Modal** - Modal/diálogo
- **Popup** - Popup
- **Skeleton** - Skeleton loader
- **Tabela** - Tabela
- **TabGroup** - Grupo de abas
- **Tag** - Tag/etiqueta
- **Tooltip** - Tooltip

### Funções

A biblioteca também inclui funções utilitárias:

- **Dialog** - Função para exibir diálogos
- **Toast** - Função para exibir notificações toast
